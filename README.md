# Ichiraku Ramen

## Features
* Create an Order
* Cancel an Order
* Add Menu Items to Order
* Customize Menu Items
* Print receipt

## Includes
* Welcome Screen - first screen the user sees allows them to start their Order
* Order Screen - Users can choose to add different menu items like, ramen, drinks, and appetizers. Users can also choose to cancel their order if they would like
* Different Item Order Screens - Each Menu Item has unique screens that prompt the user for different actions for order items
* Checkout Screen - Shows the total of the order and asks the user to confirm or cancel their order
