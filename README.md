# Ichiraku Ramen

## Includes
* 1 Parent Class MenuItem
* 3 Sub Classes Appetizer, Ramen, and Drink that extend MenuItem
* 1 Interface Discountable, that is implemented by Appetizer and Ramen.
* A file manager class that takes care of printing the users receipt after they are done ordering
* An Order class that stores menu items in a list of menu items.
* A user interface class that acts as a simple ui for the user to be able to order different menu items and get their receipt.

## Features
* A welcome screen where users can choose to stop the program or make a new order, this runs continuously on a loop until the user chooses to exit. 
* Order screen where users can choose to go back to the welcome screen, order items, cancel their order if they messed up, or checkout
* Checkout screen where users are asked if they would like to confirm or cancel their order one more time
* receipt.txt file where the users receipt is printed nicely.
